FunnelStepInput

Input for funnel step configuration. Used to define steps in a conversion funnel.

Fields

  • name
    String
    Display name of the step. Used for identification in the UI and reports. Example: "Add to Cart", "Shipping Details"
  • filters
    FiltersInput
    Filtering conditions for the step. Defines what constitutes completion of this step.
