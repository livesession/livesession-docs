Alert

Alert

  • alert_id
    string
    Unique identifier for the alert
  • name
    string
    Name of the alert
  • events
    array of object
    List of events that trigger the alert
    • kind
      integer
      Kind of event, more info [here](/docs/api/rest/EventKind)
    • value
      string
      Value or threshold that triggers the alert
  • provider
    enum
    Notification provider for the alert
    • slack
      string
    • webhooks
      string
  • slack_channel_id
    string
    Slack channel ID for notifications
  • webhook_id
    string
    Webhook ID for notifications
  • enabled
    boolean
    Whether the alert is currently active
  • creation_date
    string

    ISO 8601 timestamp when the alert was created

Example
{

  "alert_id": "",

  "name": "",

  "events": [

    {

      "kind": 0,

      "value": ""

    }

  ],

  "provider": "",

  "slack_channel_id": "",

  "webhook_id": "",

  "enabled": false,

  "creation_date": "",

  "__UNSAFE_refPath": null

}
