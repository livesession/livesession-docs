Install livesession using your package manager:

1

2
# or

React

If your React app to uses the app router, you can integrate React by creating a providers file in your app folder. This is because the @livesession/browser library needs to be initialized on the client-side using the React 'use client' directive.

Angular

LiveSession makes it easy to get data about traffic and usage of your Angular app.

Integrating LiveSession into your site enables analytics about user behavior, custom events capture, session recordings, feature flags, and more.

This guide walks you through integrating livesession into your Angular app using the JavaScript Web SDK.

Angular setup

In your src/main.ts, initialize LiveSession using your project API key and instance address. You can find both in your project settings.

Angular example
1
// src/main.ts

2
