First, install livesession using your package manager:

Install @livesession/browser
1

Angular

LiveSession makes it easy to get data about traffic and usage of your Angular app.

Integrating LiveSession into your site enables analytics about user behavior, custom events capture, session recordings, feature flags, and more.

This guide walks you through integrating livesession into your Angular app using the Browser SDK.

Angular setup

In your src/main.ts, initialize LiveSession using your project API key and instance address. You can find both in your project settings.

src/main.ts
1
import { bootstrapApplication } from '@angular/platform-browser';

2
import ls from '@livesession/browser'

3


4
import { appConfig } from './app/app.config';

5
import { AppComponent } from './app/app.component';
