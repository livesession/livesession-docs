# Web frameworks

```bash @livesession/browser installation
yarn add @livesession/browser

```

## React
If your React app to uses the app router, you can integrate React by creating a `providers` file in your app folder.
This is because the `@livesession/browser` library needs to be initialized on the client-side using the React 'use client' [directive](https://livesession.io).

```jsx React example
// src/thirdParty/livesession.jsx
'use client'

import { useEffect } from 'react'
import ls from '@livesession/browser'
import { LiveSessionProvider } from '@livesession/react'

export function WithLiveSession({ children }) {
  useEffect(() => {
    ls.init(process.env.NEXT_PUBLIC_LS_KEY, {
    })
  }, [])

  return (
    <LiveSessionProvider client={livesession}>
      {children}
    </LiveSessionProvider>
  )
}
```

## Vue.js
We use the Composition API as it provides better accessibility, maintainability, and type safety.

Start by creating a `composables` folder as well as a `useLiveSession.ts` file to that folder.
In `useLiveSession.ts`, initialize livesession as a composable:

```ts useLiveSession.ts
import ls from '@livesession/browser'

export function uselivesession() {
  ls.init('<ls_api_key>', {
  })

  return {
    ls
  }
}
````

Next, in `router/index.ts`, import the useLiveSession composable and call it.

```js router/index.ts
import { createRouter, createWebHistory } from 'vue-router'

import HomeView from '../views/HomeView.vue'
import { useLiveSession } from '@/thirdParty/useLiveSession'

const router = createRouter({
  history: createWebHistory(import.meta.env.BASE_URL),
  routes: [
    {
      path: '/',
      name: 'home',
      component: HomeView,
    },
    {
      path: '/about',
      name: 'about',
      component: () => import('../views/AboutView.vue'),
    },
  ],
})

const { ls } = useLiveSession()

export default router
````

## Angular
LiveSession makes it easy to get data about traffic and usage of your [Angular](https://livesession.io) app.

Integrating LiveSession into your site enables analytics about user behavior,
custom events capture, session recordings, feature flags, and more.

This guide walks you through integrating livesession into your Angular app using the [JavaScript Web SDK](https://livesession.io).

### Angular setup
In your `src/main.ts`, initialize LiveSession using your project API key and instance address.
You can find both in your project settings.

```js Angular example
// src/main.ts
import { bootstrapApplication } from '@angular/platform-browser';
import ls from '@livesession/browser'

import { appConfig } from './app/app.config';
import { AppComponent } from './app/app.component';

ls.init(
  '<ls_api_key>',
)

bootstrapApplication(AppComponent, appConfig)
  .catch((err) => console.error(err));
```
