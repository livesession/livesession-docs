# Rate limits
<Subtitle>
    Learn about the rate limits for the LiveSession API.
</Subtitle>

To ensure our platform remains stable and fair for everyone, some LiveSession APIs are rate-limited.
We use a variety of strategies to enforce rate limits.
We ask developers to use industry standard techniques for limiting calls, caching results, and re-trying requests responsibly.
