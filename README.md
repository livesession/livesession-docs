# LiveSession Documentation

## Local Development

> [!IMPORTANT]
> Please install Node.js (version 22.12 or higher) before proceeding.

Install dependencies:
```bash
bun add -g xyd-js
```

Dev server:
```bash
xyd
```

A local preview of your documentation will be available at `http://localhost:5175`.
